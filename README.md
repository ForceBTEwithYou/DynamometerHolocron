# DynamometerHolocron
Archive of BLE dynamometers. Self-contained Python CLIs and decoded protocol notes for force/grip devices (Muscle Meter, PitchSix Force Board, Squegg, Tindeq Progressor, VALD DynaMo): service UUIDs, packet layouts, command bytes, and quirks, alongside small bleak-based scripts to scan, connect, stream samples, and log to CSV.
